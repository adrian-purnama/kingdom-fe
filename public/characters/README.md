# Character media

Place files here named `{gender}-{role}`:

| File | Example |
|------|---------|
| Video | `male-attack.mp4`, `female-defend.mp4` |
| Poster / image | `male-attack.png`, `female-buff.png` |

- **gender:** `male`, `female`
- **role:** `attack`, `defend`, `buff`

The game profile plays the `.mp4` (looping, muted). The `.png` is used as poster and fallback if the video is missing.
